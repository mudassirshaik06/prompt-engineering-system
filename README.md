## What Is Prompt Engineering (My Definition)
Prompt Engineering is the skill of converting human intent into structured instructions that an AI can consistently follow.

A good prompt:
- Reduces ambiguity
- Sets clear roles and constraints
- Produces repeatable, high-quality outputs
