# Prompt Optimization Examples

## Example 1
Bad Prompt: “Write code for calculator”
Optimized Prompt: “You are a Python expert. Write a modular, error-handling calculator script with addition, subtraction, multiplication, and division. Include comments for each function.”

## Example 2
Bad Prompt: “Explain AI”
Optimized Prompt: “Explain AI in 200 words for 3rd-year engineering students. Use simple language, include examples, and summarize key applications.”
