# Content Generation Prompt

## ❌ Bad Prompt
Write a YouTube video script on motivation.

## ✅ Optimized Prompt
You are a professional YouTube scriptwriter.

Context:
The audience is Indian students aged 18–24 who feel confused about their future.

Instructions:
- Start with a strong emotional hook
- Use simple and relatable language
- Keep the script under 60 seconds
- End with a motivational CTA

Output Format:
- Hook
- Main message
- CTA
